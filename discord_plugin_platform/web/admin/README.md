# 後台管理應用（v1 尚未建立）

這裡預留給之後的平台操作者後台網頁——外掛審核、全域停權、配額覆蓋調整、稽核 log 查詢。

**v1 不建立這個網頁**，這些操作全部透過 `bot_integration/admin_console.py` 的終端機指令處理，理由見
`design.md` 第 3.5 節：管理功能刻意跟 `web/public/` 那個公開市集網頁完全分開，減少攻擊面。

真的需要網頁介面時，這裡會是一個**完全獨立的 FastAPI + Vue 專案**，不與 `web/public/` 共用任何程式碼、
不共用部署環境，登入機制用小範圍操作者白名單，不接一般 Discord OAuth 自助註冊。
